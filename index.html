    • pyhenkan

    Transcoding suite built around VapourSynth and FFmpeg

    About

    pyhenkan is a transcoding suite written in Python and built around VapourSynth and FFmpeg. It is an effort to provide an alternative to MeGUI and StaxRip on Linux.

    Features

    Input

    • Handles all formats supported by FFmpeg
    • Batch transcodes files with the same structure
    • Uses a job control system to queue transcoding tasks

    Output

    • Video: Pasthrough, x264, x265, VP8, VP9
    • Audio: Passthrough, AAC (native, faac, fdk-aac), Flac, Mp3, Opus, Vorbis
    • Subtitle: Passthrough
    • Video filters: Resize, Denoise, Deband, Resample, Color matrix, Trim
    • Audio filters: Channel conversion, Resample, Trim
    • Containers: MKV only (for now)
    • Handles track title, language and default flag

    Chapter editor

    • Accepts timecodes or frame numbers (CFR videos only) as boundaries
    • Dynamically converts between timecodes and frame numbers
    • Supports Ordered Chapters/Segment Linking
